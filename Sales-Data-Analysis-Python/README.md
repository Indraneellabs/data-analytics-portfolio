# Sales Data Analysis & Customer Segmentation

## Overview
This project analyzes sales data to identify trends, top-performing products, and customer behavior.

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib

## Key Steps
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Customer Segmentation
- Data Visualization

## Key Insights
- Top 10 customers contribute ~29.5% of total revenue
- Certain months show peak sales trends
- Product categories like Classic Cars dominate revenue

## Outcome
Derived actionable insights to support business decision-making.
