# Customer Churn Prediction

## 📌 Overview
This project builds a machine learning model to predict customer churn (whether a customer will leave a service).

## 🎯 Objective
To identify customers likely to churn and help businesses take proactive actions to improve retention.

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Scikit-learn
- Logistic Regression

---

## 🔄 Workflow
1. Data Cleaning and Preprocessing  
2. Handling Missing Values  
3. Encoding Categorical Variables (One-Hot Encoding)  
4. Model Training (Logistic Regression)  
5. Model Evaluation  

---

## 📊 Model Performance
- Accuracy: ~64%  
- Recall (Churn): ~92%  
- Improved recall using class balancing and threshold tuning  

---

## 💡 Key Insights
- Model performs well in identifying churn customers  
- Recall is prioritized over accuracy for business impact  
- Useful for customer retention strategies
